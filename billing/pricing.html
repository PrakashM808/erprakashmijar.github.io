FREQUENTLY ASKED QUESTIONS
Can I scan any Linux server? ▶
Yes — you can scan any Linux machine via SSH. You provide the IP, username, and password (or SSH key). The scanner connects via SSH and reads port status, SSH config, auth logs, package versions, and file permissions. Works on Ubuntu, Debian, CentOS, RHEL, and most Linux distributions.
Is my server data safe? ▶
Your scan data is processed by your own backend API — either running locally or on a server you control (Railway, Render, VPS). No scan results are stored on our servers. SSH credentials are never logged and only used for the scan connection.
What payment methods are accepted? ▶
Via Stripe: all major credit and debit cards (Visa, Mastercard, Amex). Via Lemon Squeezy: cards + PayPal + local payment methods. Lemon Squeezy also handles VAT and sales tax automatically — ideal if you're outside the US.
Can I cancel anytime? ▶
Yes — cancel anytime from your dashboard. Your plan stays active until the end of the billing period, then downgrades to Free automatically. No cancellation fees.
Do I need to install anything? ▶
The dashboard frontend is just a website — no installation needed. To get live scan data (vs demo data), you run the Python backend on your machine or deploy it to Railway/Render. Setup takes about 5 minutes following the README.
What is network discovery? ▶
Network discovery pings all hosts on your local subnet (e.g. 192.168.1.0/24) to find live devices. It shows their IP, open ports, and whether SSH is available. You can then click any discovered device to run a full security scan via SSH.
